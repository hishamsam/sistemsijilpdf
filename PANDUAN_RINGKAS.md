# Panduan Ringkas Sistem Sijil

## Untuk Pengguna Baru - 5 Langkah Mudah

### Langkah 1: Muat Turun & Extract
```
1. Muat turun fail SistemSijil_v1.0.0.zip
2. Klik kanan â†’ Extract All â†’ Extract
3. Buka folder yang di-extract
```

### Langkah 2: Jalankan Aplikasi
```
1. Double-click pada sistem_sijil.exe
2. Jika ada amaran Windows:
   - Klik "More info"
   - Klik "Run anyway"
```

### Langkah 3: Tetapan Awal
```
1. Pergi ke Tetapan (Settings)
2. Pilih bahasa: BM atau EN
3. (Pilihan) Set kata laluan untuk keselamatan
```

### Langkah 4: Cipta Program Pertama
```
1. Klik "Program Baru" di Dashboard
2. Isi maklumat program:
   - Nama Program
   - Kod Program
   - Tahun
   - Tarikh
3. Klik "Simpan"
```

### Langkah 5: Tambah Peserta & Jana Sijil
```
1. Buka program yang dicipta
2. Klik "Import Excel" atau tambah manual
3. Pilih peserta â†’ Klik "Jana Sijil"
4. Sijil PDF akan dijana!
```
