FAQ

Soalan Lazim

Jawapan kepada soalan yang sering ditanya

Q: Adakah aplikasi ini memerlukan internet?

Tidak. Sistem Sijil berfungsi 100% offline. Semua data disimpan secara lokal di komputer anda. Internet hanya diperlukan untuk muat turun aplikasi sahaja.

Q: Bolehkah saya guna di Mac atau Linux?

Buat masa ini, tidak. Aplikasi ini hanya untuk Windows 10/11 (64-bit). Versi Mac dan Linux mungkin akan dibangunkan pada masa hadapan jika ada permintaan.

Q: Adakah data saya selamat?

Ya. Semua data disimpan dalam database SQLite di komputer anda sendiri. Tiada data dihantar ke mana-mana server. Anda juga boleh backup data ke fail JSON bila-bila masa.

Q: Berapa banyak sijil boleh dijana?

Tiada had. Anda boleh menjana seberapa banyak sijil yang diperlukan tanpa sebarang batasan.

Q: Format fail Excel yang disokong?

.xlsx dan .xls - Format Microsoft Excel standard. Pastikan lajur pertama mengandungi nama peserta.

Q: Bolehkah saya edit template sijil?

Ya. Anda boleh menggunakan template Custom untuk menyesuaikan rekabentuk sijil mengikut keperluan organisasi anda.

Q: Aplikasi tidak boleh dibuka / error?

Pastikan Visual C++ Redistributable sudah dipasang. Jika masih bermasalah, cuba run sebagai Administrator atau hubungi kami untuk bantuan.

Mengapa Perlu Reset?

Aplikasi ini datang dengan data contoh (sample data) untuk tujuan demo. Sebelum anda mula menjana sijil sebenar, anda WAJIB memadam data contoh ini terlebih dahulu untuk mengelakkan kekeliruan.

1

Buka Menu Tetapan

Klik pada menu "Tetapan" atau "Settings" di bahagian navigasi aplikasi.

2

Cari Bahagian "Data & Sandaran"

Scroll ke bawah sehingga anda jumpa bahagian "Data & Sandaran" atau "Data & Backup".

3

Klik "Reset Semua Data"

Klik butang "Reset Semua Data" berwarna merah. Satu popup pengesahan akan muncul.

‚ö†Ô∏è Amaran: Tindakan ini akan memadam SEMUA data termasuk program, peserta, dan sijil. Pastikan anda sudah backup jika ada data penting.
4

Sahkan Reset

Pada popup pengesahan, klik "Ya, Reset" atau "Yes, Reset" untuk mengesahkan tindakan.

‚úì

Sedia Untuk Mula! üéâ

Data contoh telah dipadam. Anda kini boleh mula mencipta program baru dan menjana sijil sebenar.

Setup Pengesahan Sijil Online

Bina sistem pengesahan sijil online menggunakan Google Sheets + Apps Script (100% PERCUMA)

üí°

Gambaran Keseluruhan

Sistem ini membolehkan orang awam mengesahkan sijil melalui halaman web tanpa perlu memasang aplikasi. Data sijil disimpan dalam Google Sheets sebagai database online.

üì§ Export CSV dari App ‚Üí üìä Google Sheets ‚Üí ‚ö° Apps Script API ‚Üí üåê Halaman Web
1

Export Data Sijil dari App

  1. 1. Buka aplikasi Sistem Sijil
  2. 2. Pergi ke Tetapan (Settings)
  3. 3. Cari bahagian "Pengesahan Online"
  4. 4. Klik butang "Export Data Sijil"
  5. 5. Pilih folder untuk simpan
üìÅ 4 fail akan dicipta: sijil_data.csv, apps_script.js, verify.html, README.txt
2

Buat Google Sheets & Import CSV

  1. 1. Buka sheets.google.com
  2. 2. Klik "+ Blank" untuk spreadsheet baru
  3. 3. Namakan: "Database Sijil"
  4. 4. Klik File ‚Üí Import ‚Üí Upload
  5. 5. Pilih fail sijil_data.csv
  6. 6. Import location: Replace current sheet
  7. 7. Klik "Import data"
3

Setup Google Apps Script

  1. 1. Dalam Google Sheets, klik Extensions ‚Üí Apps Script
  2. 2. Padam kod sedia ada (Ctrl+A, Delete)
  3. 3. Buka fail apps_script.js dengan Notepad
  4. 4. Copy semua kod dan paste dalam Apps Script
  5. 5. Klik Save (üíæ)
  6. 6. Namakan projek: "Pengesahan Sijil API"
‚ö†Ô∏è Jangan tekan butang "Run" dalam editor.
4

Deploy Web App

  1. 1. Klik "Deploy" ‚Üí "New deployment"
  2. 2. Klik ikon ‚öôÔ∏è ‚Üí Pilih "Web app"
  3. 3. Execute as: Me
  4. 4. Who has access: Anyone ‚ö†Ô∏è PENTING!
  5. 5. Klik "Deploy"
  6. 6. Klik "Authorize access" ‚Üí Allow
  7. 7. SALIN URL yang dipaparkan
üìã URL seperti: https://script.google.com/macros/s/AKfycbx.../exec
5

Setup Halaman Web Pengesahan

  1. 1. Buka fail verify.html dengan Notepad
  2. 2. Cari baris: const APPS_SCRIPT_URL = 'YOUR_APPS_SCRIPT_URL_HERE';
  3. 3. Gantikan YOUR_APPS_SCRIPT_URL_HERE dengan URL dari Langkah 4
  4. 4. Simpan fail (Ctrl+S)

Test Pengesahan

  1. 1. Double-klik fail verify.html
  2. 2. Masukkan nombor sijil (contoh: CERT-PROG-2024-0001)
