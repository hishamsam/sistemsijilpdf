INSTALLATION:
=============

1. EXTRACT ALL FILES
   - Right-click on this ZIP file
   - Select "Extract All..."
   - Choose location (Desktop or Program Files)
   - Click "Extract"

2. RUN THE APPLICATION
   - Open the extracted folder
   - Double-click on "sistem_sijil.exe"

3. IF WINDOWS WARNING APPEARS
   - Click "More info"
   - Click "Run anyway"
   - This is normal for new applications

SYSTEM REQUIREMENTS:
====================
- Windows 10 or Windows 11 (64-bit)
- RAM: 4 GB minimum
- Disk space: 200 MB

KEY FEATURES:
=============
✓ Create and manage certificate programs
✓ Import participant list from Excel/CSV
✓ Generate PDF certificates with QR code
✓ Offline/online certificate verification
✓ Backup and restore data
✓ Bilingual (Malay & English)

IMPORTANT FILES (DO NOT DELETE):
================================
- sistem_sijil.exe      → Main application
- flutter_windows.dll   → Runtime
- pdfium.dll           → PDF engine
- sqlite3.x64.windows.dll → Database
- data/                → Assets folder

DATA LOCATION:
==============
Your data is saved at:
C:\Users\[Username]\Documents\SistemSijil\

TROUBLESHOOTING:
================

Problem: "Windows protected your PC"
Solution: Click "More info" → "Run anyway"

Problem: Antivirus blocking
Solution: Add folder to exclusion list

Problem: Missing DLL error
Solution: Re-extract ALL files from ZIP
